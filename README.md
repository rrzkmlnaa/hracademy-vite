## Getting Started

### Install

Create the project.

```bash
git clone https://github.com/rrzkmlnaa/hracademy-vite.git hracademy
```

Access the project directory.

```bash
cd hracademy
```

Install dependencies.

```bash
pnpm install
```

Serve with hot reload at <http://localhost:5173>.

```bash
pnpm run dev
```

### Build

```bash
pnpm run build
```
